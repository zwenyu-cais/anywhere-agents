# ci-mockup-figure

Build HTML mockups of systems, dashboards, and timelines, then capture them as space-efficient PNG / PDF figures via headless Chrome. Includes an abstract-diagram path (TikZ, skia-canvas, Illustrator ExtendScript) for architecture figures with arrow routing between non-adjacent nodes where HTML / CSS fails.
